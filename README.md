# Recycling Production Line Manager System

This project is a backend-focused system designed to manage and simulate
data for a recycling production line.

## Features
- SQL database schema for production line management
- Fake data generation using Faker.js
- Sample JSON data for testing and development
- AI prompts for future automation and analysis

## Tech Stack
- Node.js
- Faker.js
- SQL

## Project Structure
- `database/` → schema and data generation scripts
- `ai-prompts/` → prompts for AI-based automation
- `sample-candidates.json` → generated test data

## How to Run
```bash
npm install
node database/generateCandidates.js
